# 🚑 AI Injury Detection & First Aid Recommendation System

An AI-powered healthcare assistance system that analyzes injury symptoms and injury images to provide preliminary injury assessment, risk evaluation, and first-aid recommendations using Google Gemini AI.

## 📖 Overview

Many people struggle to determine the seriousness of an injury and the immediate first-aid measures they should take before consulting a medical professional. This project aims to provide quick and accessible guidance by leveraging Artificial Intelligence.

The system allows users to:

* Analyze injuries based on symptoms
* Upload injury images for visual assessment
* Receive risk-level evaluation
* Get first-aid recommendations
* Understand when medical attention may be required

> **Note:** This application is intended for educational and demonstration purposes only and is not a substitute for professional medical advice.

## 🧠 How It Works

### Symptom Analysis

1. User answers symptom-related questions.
2. Responses are collected in JavaScript.
3. Data is sent to Flask backend in JSON format.
4. Flask creates a prompt for Gemini AI.
5. Gemini analyzes symptoms and generates:

   * Likely injury
   * Risk level
   * First-aid advice
   * Medical recommendation

### Image Analysis

1. User uploads an injury image.
2. Image is processed using Pillow.
3. Flask sends the image to Gemini Vision AI.
4. AI analyzes visual indicators such as:

   * Swelling
   * Bruises
   * Cuts
   * Visible trauma
5. Results are displayed to the user.
